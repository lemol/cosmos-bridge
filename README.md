## Development

**Clone**

```
git clone https://github.com/lemol/cosmos-bridge
cd cosmos-bridge
```

**Install**

```
yarn
```

**Run the server**

```
cd server && yarn dev
```

**Run the client**

```
cd client && yarn dev
```
