## Development

**Clone**

```
git clone https://github.com/lemol/cosmos-bridge
cd cosmos-bridge
```

**Install**

```
yarn --cwd server install
yarn --cwd client install
```

**Run the server**

```
yarn --cwd server dev
```

**Run the client**

```
yarn --cwd client dev
```
